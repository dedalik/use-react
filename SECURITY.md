# Security Policy

## Supported Versions

Only the latest published major version is actively supported with security fixes.

## Reporting a Vulnerability

Please report security issues privately before creating a public issue.

- Open a private security advisory on GitHub for this repository, if available.
- If private advisories are not available, open an issue with minimal reproduction details and request a private contact channel.

Please include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Suggested remediation (optional)

We will acknowledge reports as soon as possible and coordinate disclosure after a fix is prepared.
